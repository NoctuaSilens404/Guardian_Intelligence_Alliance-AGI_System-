# GIA - Guardian Intelligence Alliance

## Especificación del Hardware

### Casco Inteligente
- **MCU**: ESP32
- **Sensores**:
  - Sensor de pulso cardíaco (pin 34)
  - Sensor de temperatura (pin 35)
  - Sensor SpO2 (pin 32)
  - Sensor de caída / acelerómetro (pin 33)
  - Botón de pánico (pin 25)
- **Actuadores**:
  - LED RGB (pines 26, 27, 14)
  - Buzzer (pin 12)
- **Comunicación**: WiFi + MQTT

### Chaleco Inteligente
- **MCU**: ESP32
- **Sensores**:
  - Sensor de pulso cardíaco (pin 34)
  - Sensor de temperatura (pin 35)
  - Sensor SpO2 (pin 32)
  - Botón de pánico (pin 25)
- **Comunicación**: WiFi + MQTT
